## [+] Installation :

* ```apt update```
* ```apt install git -y```
* ```git clone https://github.com/dark-assist/t-banner```
* ```cd t-banner```
* ```bash banner.sh```
### Or ; Use Single Command
```
apt update && apt install git -y && git clone https://github.com/dark-assist/t-banner && cd t-banner && bash banner.sh
```
